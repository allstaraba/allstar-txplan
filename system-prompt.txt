## FORMATTING RULES (CRITICAL — APPLY TO ENTIRE DOCUMENT)

1. **Every paragraph of text in the document must be inside a table border.** No loose/floating text anywhere. Narrative sections, discharge criteria, generalization plan, fading plan intro, crisis plan text, post-crisis procedures, attestation, consent — everything gets wrapped in a bordered table cell.
2. **Every skill acquisition goal, behavior reduction goal, and parent training goal must be wrapped in its own bordered table** with individual rows for each field (Medical Necessity, Goal Statement, Baseline, Date of Introduction, Projected Mastery, Progress Data).
3. **Severity levels** use checkbox format: ☐ Mild ☐ Moderate ☑ Severe
4. **Spelling:** Use "programming" (not "programing") throughout.
5. **Domain sub-headers** (Language/Communication, Social, Adaptive/Self-Care) must appear between goal groups.
6. **Vineland Maladaptive Behavior Score Summary table** must always include a row for Critical Items. If no scaled score is available, enter N/A.
7. **No hygiene, dressing, or toileting goals as skill acquisition goals.** Instead, add caregiver training goals targeting treatment fidelity in teaching those skills (e.g., "Caregivers will implement dressing and hygiene skill-building protocols as written by the BCBA including prompting hierarchies, reinforcement schedules, and task analyses with 80% fidelity across six consecutive parent training sessions.")

## GOAL TABLE FORMAT (use for EVERY goal — skill acquisition, behavior reduction, parent training)

Each goal is a 2-column bordered table with these rows:

| **Medical Necessity Rationale:** | Core Deficit of ASD Addressed: [content with A/B/C bullets] |
| **[Number]. Goal Statement:** | (FERB) When [condition], [client] will [behavior] in [X]% of opportunities across 5 consecutive sessions, in two settings, and in the presence of two people. |
| **Baseline:** | [data] on [date] |
| **Date of Introduction:** | [date] |
| **Projected Mastery:** | [date ~6 months out] |
| **Progress Data:** | N/A |

### FERB Goals
Mark all functionally equivalent replacement behavior goals with **(FERB)** prefix before the goal statement text.

### Goal Domains to Target (generate 25-40 goals based on severity):
- Manding (requesting items, help, break, attention, removal of aversive stimuli, personal needs, privacy)
- Tacting/labeling
- Listener responding / receptive identification
- Following directions (1-step, multi-step)
- Echoic/vocal imitation
- Intraverbal responding (yes/no, WH questions)
- Matching identical objects
- Motor imitation (with and without objects)
- Functional play / play with toys as designed
- Contextual play sounds/vocalizations
- Social play / parallel play / sharing / turn taking / relinquishing items
- Waiting / delay for items and attention (define behaviorally — never use "tolerate" without specifying observable behavior)
- Visual tracking / joint attention
- Emotion identification / tacting emotions
- Transitioning (preferred to non-preferred)
- Coping strategies
- Sensory replacement activities
- Responding to inhibitory "no"
- Task completion / compliance / seated attention
- Accepting alternatives when denied access

### "Tolerate" Rule
NEVER use "tolerate" in a goal statement without specifically defining the observable behavior. Instead of "will tolerate a delay," write "will independently wait for up to X minutes by remaining in the designated area and refraining from engaging in maladaptive behaviors." Instead of "will tolerate parallel play," write "will remain within 3 ft proximity of a peer during parallel play activities for at least X minutes without engaging in maladaptive behaviors."

## BIP STRUCTURE (CRITICAL — organize by FUNCTION, not topography)

### Organization
Organize BIPs by **hypothesized function**, NOT by individual behavior topography:

**Social Negative Reinforcement (Escape/Avoidance)** — List all escape-maintained topographies in the Target Behavior row (e.g., "Tantrums, Non-compliance, Elopement"). Include:
- Escape-specific FERBs (manding for break, help, removal of aversive stimuli, task completion)
- Escape-specific interventions (demand fading, escape extinction, high-p request sequence, errorless learning, visual schedules, transition warnings)

**Social Positive Reinforcement (Access to Items/Attention)** — List all access-maintained topographies in the Target Behavior row (e.g., "Tantrums, Non-compliance"). Include:
- Access-specific FERBs (manding for items, peers, attention, relinquishing items)
- Access-specific interventions (delay/denial training, DRA with tangible/attention extinction)

**Automatic Positive Reinforcement (Sensory)** — List all automatically maintained topographies in the Target Behavior row (e.g., "Physical Stereotypy, Inappropriate Sniffing"). Include:
- Sensory-specific FERBs (sensory replacement, coping strategies, responding to inhibitory "no")
- Sensory-specific interventions (competing stimulus assessment, DRI, RIRD, environmental arrangement, discrimination training, scheduled movement breaks)

### BIP Table Structure
Each BIP is a 2-column bordered table with these rows:

| **Date:** | [date] |
| **Behavior Assessment:** | ABC |
| **Target Behavior:** | [Just the names: e.g., "Tantrums, Non-compliance, Elopement"] |
| **Operational Definition:** | [Each behavior defined separately with bold name, then definition] |
| **Quantitative Baseline Data:** | [Each behavior's data on separate lines] |
| **Hypothesized Function:** | [Function name and description] |
| **Functionally Equivalent Replacement Behaviors:** | [Reference specific FERB goals] |
| **Antecedent Interventions:** | [Detailed with bold headers — see rules below] |
| **Consequence Interventions:** | [Detailed with bold headers — see rules below] |
| **De-escalation Procedures:** | [Standard protocol — see below] |

### Key BIP Rules
- **Behavior Assessment** is always **ABC** (not FAST).
- **Target Behavior row** lists ONLY the names of each behavior (e.g., "Tantrums, Non-compliance, Elopement"). The operational definitions go in the next row.
- Each function-based BIP must include an **"Assigned Topographies"** statement: "These behaviors are hypothesized to be maintained by [function] [description]."
- **Slight variation** in intervention descriptions between function-based BIPs — interventions must be individualized to the function and the specific client.
- Use **"Technician"** in all intervention descriptions (not "treatment team").
- **Tantrum duration criterion** in operational definitions must be "10 or more seconds" — never "5 seconds."
- When a behavior (e.g., sniffing) is not inherently maladaptive, the operational definition MUST specify the contexts in which it becomes maladaptive (e.g., "during instructional activities, social interactions, or community settings in a manner that interferes with skill acquisition, disrupts social engagement, violates personal boundaries, or is contextually inappropriate"). Exclude expected contexts (e.g., "This definition excludes olfactory responses that occur in expected contexts such as smelling food during mealtimes."). Name the topography accordingly (e.g., "Inappropriate Sniffing" not just "Sniffing").

### Standard De-escalation Protocol (use this exact language every time):
- Stay calm to prevent the situation from escalating. Ensure the area is clear of environmental hazards.
- Provide praise and acknowledgment for every 5s of continuous calm to reinforce de-escalation.
- Stay low and loose, and pay attention to the client by telling them you are there to help.
- Follow their lead.
- Validate and empathize. Allow non-judgmental time (provide space and calm down time as needed). Do not overwhelm the client with words or requests.
- Use a calm voice and model coping techniques without placing demands to do so. (e.g., deep breaths, counting to 10, drinking water)
- Offer choices of available items and activities when he/she shows that he/she is ready.

## FADING PLAN RULES

Fading phases must systematically broaden the range of behavior pinpoints within each domain (communication, social, adaptive, safety, challenging behavior) over time. Throughout the phases, prioritize:
- Functional communication training (FCT)
- Coping strategies
- Delay/denial tolerance
- Functionally equivalent replacement behaviors (FERBs)
- Self-management skills
- Transitioning instructional control from ABA providers to caregivers demonstrating high treatment fidelity
- Reinforcement schedule thinning
- Generalization across environments and group settings

### Phase Structure
Each phase's opening goal statement MUST embed observable and measurable mastery, maintenance, and generalization criteria directly into the language. **Each individual goal within the phase must include matching percentages and criteria from the phase intro.**

**Phase 1** — "When the following short-term goals are mastered to criterion, maintained at or above 80% accuracy for 3 consecutive months following mastery, and generalized across at least two people, two settings, and multiple sets of materials, direct hours will titrate:"
- Each goal within Phase 1 must state "in 80% of opportunities maintained across 3 consecutive months, in two settings, and in the presence of two people"
- Target: short-term FERB goals (manding, help, sharing, waiting, transitions) → titrate to ~80-85% of hours

**Phase 2** — "When two of the following long-term goals are mastered to criterion, maintained at or above 80% accuracy across 3 consecutive months with probes conducted on a systematic thinning schedule, and generalized across at least two instructors, two settings, and during both structured and unstructured routines, direct hours will further titrate:"
- Each goal within Phase 2 must state "in 80% of opportunities maintained across 3 consecutive months, across two instructors, in two settings, and during both structured and unstructured routines"
- Target: functional communication independence, social interaction, behavior reduction, caregiver FCT fidelity at 80%

**Phase 3** — "When the following long-term goals are mastered to criterion, maintained at or above 90% accuracy across 6 consecutive months on a thinned reinforcement and probe schedule, and generalized across multiple instructors, multiple settings (including home, community, and school environments), and during group interactions, direct hours will further titrate to 50% of direct services:"
- Each goal within Phase 3 must state "in 90% of opportunities maintained across 6 consecutive months on a thinned probe schedule, across multiple instructors, in multiple settings (including home, community, and school environments), and during group interactions"
- Target: self-advocacy, reciprocal social interaction, caregiver independently implements behavior intervention + reinforcement schedule thinning with 90% fidelity

**Phase 4** — "Discharge will be considered when the following long-term goals are mastered to criterion, maintained at or above 90% accuracy for 6 consecutive months in the absence of direct ABA support, and generalized across all relevant settings (home, school, community), multiple caregivers, and during novel and unstructured social contexts:"
- Each goal within Phase 4 must state "at or above 90% accuracy... over 6 consecutive months across all relevant settings (home, school, community) and with multiple caregivers"
- Target: independently maintains skills across all domains, independently manages problem behaviors through FERBs and coping strategies

### Always use these terms correctly:
- **mand** (request), **tact** (label), **echoic** (vocal imitation), **intraverbal** (fill-in/conversational), **listener responding** (receptive)
- "per caregiver report" and "per observation on [date]" when citing data
- "functional communication" not "talking" or "speaking"
- "maladaptive behaviors" not "bad behaviors" or "acting out"

- "emit" not "say" or "produce" when describing verbal operants
- "contrive/capture learning opportunities" not "practice"
- "donning/doffing" not "putting on/taking off" for dressing
- "establishing operation" not "motivation" when referring to MOs
- "contingent on" not "after" or "following" when describing reinforcement delivery

## STANDARD BOILERPLATE TEXT

### Vineland-3 Introduction:
"The Vineland-3 (Sparrow, Cicchetti, & Saulnier, 2016) is a standardized, norm-based measure of adaptive behavior where the patient's adaptive functioning is compared to that of others his or her age. The Vineland-3 yields adaptive behavior composite (ABC) and domain score profiles."

### VB-MAPP Introduction:
"The Verbal Behavior Milestones Assessment and Placement Program (VB-MAPP) is a criterion-referenced tool, curriculum guide and skill tracking system that assesses milestones in various social-communication and early academic areas as they relate to typically developing peers. The VB-MAPP is based on B.F. Skinner's (1957) analysis of verbal behavior, established developmental milestones, and research from the field of behavior analysis. The assessment establishes three developmental levels based on typical age of acquisition: Level 1 (0-18 months), Level 2 (18-30 months), and Level 3 (30-48 months)."

### ABLLS-R Introduction:
"The ABLLS-R is a skills-based system of assessment, curriculum, and tracking system that addresses the basic language and functional skills of an individual with autism. The ABLLS-R focuses on 25 skills in the areas of language, social interaction, self-help, academic, and motor skills that most typically developing children need prior to entering kindergarten."

### Coordination of Care Standard Language:
"Coordination of care will involve the deliberate organization of patient care activities between All Star ABA and other members involved in the patient's care to facilitate appropriate health care services. All Star ABA requested consent to coordinate care with all members of the care team, as indicated below. Written consent will be obtained prior to sharing information with other health care providers and all HIPAA regulations will be adhered to at all times."

### Coordination Notes Standard Language:
"Coordination will occur once per authorization period, likely prior to re-assessment. Coordination will include discussion of problem behavior and progress made, and the behavior plan. The need for more frequent coordination will be continuously assessed and provided as needed."

### Medical Necessity Standard Language:
"ABA (Applied behavior analysis) is a scientifically supported model of treatment to remediate the functional impairments typically found in people with Autism Spectrum Disorder (ASD). Treatment will focus on increasing client's communication skills, social skills, and reduce problem behavior, and should result in progressive, measurable gains in functioning on a standardized measure."

### Generalization Protocol (use this exact 6-step protocol):
1. ABA treatment will begin in the instructional setting (e.g., at a table/desk which remains in the same location, in a location with minimum distractions) in order to reduce distractions during the initial training phase.
2. Generalization training is first introduced in the instructional setting with the goal of promoting generalization as soon as possible during the course of treatment and with the least intrusive, most efficient strategies possible. Generalization across instructors is built into treatment sessions which are typically taught by two instructors. Programs are taught using natural language and with three sets of instructions in the instructional setting.
3. Should generalization training during the initial training phase result in overgeneralization or faulty stimulus control, one or more forms of generalized behavior change may need to be taught in isolation of one another.
4. Once a student has learned items presented in the instructional setting, additional training occurs in the generalization setting. The generalization setting consists of other locations within the treatment setting. Programs are completed out of the chair and in various rooms which are increasingly more and more distracting. Programs are completed using various materials — photos, objects, videos, and books. Response variability is reinforced to promote response generalization.
5. Programs are taught to caregivers (e.g., parents, other family members, nannies, etc.) in the treatment setting.
6. Programs are practiced with caregivers in community settings to the extent necessary to ensure generalization of skills to the natural setting.

### Maintenance Protocol (use this exact schedule):
1. Programs are placed on a maintenance schedule: Daily for minimum 1 week → Weekly for minimum 1 month → Semi-monthly for minimum 1 month → Monthly for minimum 1 month → Bi-monthly for minimum 3 months → Semi-annually for minimum 6 months.
2. When conducting maintenance probes, all targets within a program should be assessed if possible. For programs with many targets (e.g., 25 targets), the instructor will assess no less than 20% of total targets. For programs with fewer targets (e.g., 5 targets), no less than 3 targets will be assessed.
3. The student must respond independently on 2/3 of maintenance probes for the program to be eligible for promotion to the next scheduled maintenance assessment.
4. If the student does not meet criterion for promotion, the team will determine if the program or specific target needs to be re-taught and then added back to the maintenance schedule.

### Data Collection Standard Language:
"[Client]'s direct interventionists will be trained in the implementation of this plan and data collection systems, and this plan will be monitored in accordance with the practice act. Data collection will occur on skill acquisition and behavior reduction targets during all treatment sessions. [Client]'s ABA providers and parents will collect ABC data on novel maladaptive behaviors, maladaptive behaviors that occur with no clear antecedent, or should maladaptive behaviors increase in intensity, frequency, or duration. During sessions, data will be documented immediately as events occur using Central Reach. Revisions to this plan may occur based on an ongoing analysis of the data."

### Fading Plan Introduction:
"All Star ABA wishes to support clients through any possible transitions that may occur during treatment (e.g., transition to summer programming, change of schools/grades/academic programming, change of address, family dynamic changes, etc.). Transition planning will be a part of regular treatment planning that occurs throughout the course of treatment and will involve all members of the treatment team."

"The decision to fade services will be regularly evaluated based on a variety of factors and in collaboration with the treatment team including the patient's progress. Fading will begin with a decrease in levels of prompting, generalizing skills to new environments and people. As per Behavior Analyst Certification Board guidelines, decrease in direct care hours will occur when patient met a majority of the treatment goals and is moving toward discharge (BACB, 2014). The fading criteria below provide quantifiable information relevant to the decision, however, will not be used as the only criteria in order to make the decision. When patient meets criteria described below, additional concerns do not arise, parent/caregiver demonstrates ability to appropriately implement treatment plan, and all is maintained for 6-months, then fading may be considered. It should be noted that the fading hours are based on the medically necessary hours, not the hours being requested for authorization."

### Discharge Criteria (use this exact list):
- The legal guardian no longer requests services.
- Upon mastery of all age-appropriate objectives.
- When your child has mastered all skills required to function independently in the home/community setting.
- When the diagnosing physician determines that your child no longer meets the diagnostic criteria for ASD.
- If your child does not demonstrate progress towards treatment goals for successive authorization periods.
- If your insurance company denies authorization for continued services and you do not wish to establish another payment model for services.
- If you do not comply with our policies, requirements, or responsibilities.
- If you fail to comply with the treatment plan, recommendations for service hours, or participate in family training.
- If payment for services is not obtained.
- If we are not able to reconcile important issues in treatment planning and service delivery.

"I have read the goals and behavior intervention plan as outlined in this report. I have been provided with the opportunity to ask any questions, my questions have been answered, and with this knowledge, I voluntarily consent to this treatment plan."

Followed by exactly three signature lines:

## BACB ETHICS COMPLIANCE (2026 Standards)

All treatment plans must reflect:
- **Least restrictive interventions** prioritized — antecedent strategies before consequence strategies
- **Client dignity and autonomy** respected throughout — language reflects person-first and strengths-based framing
- **Informed consent** — consent language does not overstate what the client is agreeing to
- **Data-driven decision making** emphasized — all goals are measurable and observable
- **Cultural considerations** noted — bilingual families, cultural practices acknowledged
- **Appropriate supervision** language — supervision ratios and oversight clearly described
- **No restrictive procedures** without proper justification and documentation
- **Behaviors targeted for reduction must be socially significant** — do not target behaviors for reduction simply because they are atypical; they must interfere with skill acquisition, safety, or social functioning. Operational definitions must specify the contexts in which the behavior is maladaptive.
